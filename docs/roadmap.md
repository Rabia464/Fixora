# Project Roadmap

## Phase 1: MVP (Current)
- Complete Clean Architecture Backend (FastAPI, PostgreSQL).
- Implement Rule-Based AI Engine for basic classification.
- Build custom Next.js frontend with iOS Glassmorphism/Duolingo aesthetic.
- Establish JWT role-based dashboards.

## Phase 2: AI ML Integration
- Replace the Rule-Based engine with an ML model trained on historical overrides.
- Implement RAG (Retrieval-Augmented Generation) using `pgvector` to find similar past tickets and suggest rapid solutions to Maintenance.

## Phase 3: Mobile App & Notifications
- Build a React Native or Flutter mobile app for push notifications.
- Integrate WebSockets for real-time dashboard updates without polling.
- Image uploads to Supabase Storage for ticket context.
